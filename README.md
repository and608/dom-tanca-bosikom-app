# Дом танца Босиком

Мобильное приложение под Android для автоматизации деятельности детского танцевального коллектива «Дом танца Босиком».

## Назначение приложения
Приложение предназначено для упрощения управления основными процессами коллектива, включая расписание, учет посещаемости, контроль оплаты и информирование родителей.

## Основные функции
- авторизация пользователей;
- просмотр расписания занятий;
- учет посещаемости;
- контроль статуса оплаты;
- публикация новостей;
- просмотр уведомлений.

## Используемые технологии
- Kotlin
- Android Studio
- Firebase Authentication
- Firebase Realtime Database

## Назначение проекта
Проект разработан в рамках выпускной квалификационной работы и практики.
